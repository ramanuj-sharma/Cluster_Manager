# Run the App

## Create Python environment
### Windows
	python -m venv .venv
### Linux/Mac
	python -m venv .venv

## Activate Environment
### Windows
	.venv\Scripts\activate
### Linux/Mac
	. .venv/bin/activate

## Install Flask
First time running, install Flask

	pip install Flask

Also install all other dependencies.

## Run the App
	python server.py
